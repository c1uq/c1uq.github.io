Purpose

Buford was originally a project meant to do various mathematical functions. I eventually decided to create an algebra bot (bu.bot.alge) inside this library, but it was very poorly written so hence: Buford2. Buford's main functionality will be to solve algebraic equations; it will be able to determine what tools to use and how to use them to get the desired output. This JavaScript program will have a general function which tries to detect the problem and how to solve it, and more specified user-friendly functions if you already know what needs to be done to solve the problem and don't want to use extra processes. While still incomplete, the code I have written so far was cloned here.
